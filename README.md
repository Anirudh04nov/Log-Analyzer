# 🔐 Log Analyzer Dashboard

A Python-based cybersecurity tool that analyzes system logs to detect brute-force attacks and suspicious login activity.

---

## 🚀 Features

- Detects brute-force attacks
- Identifies suspicious login attempts
- Displays alerts with severity levels
- Shows attack timeline
- Visualizes top attackers using graphs

---

## 🛠️ Tech Stack

- Python
- Flask
- Matplotlib
- Regex

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python app.py
